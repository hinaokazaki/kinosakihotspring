SEO Best Practices

1. Tags

Title tags - the title tag tells search engines what the page is about and that the page on your website is relevant for that keyword or keyword phrase. It should be unique for every page.

H1 tags, H2 tags - A header tags are much like the subject line of your web page. you should only use keyword once in the h1 tag and if there are multiple sections you can also use h2 tag.

2. Images

Optimise your image - To make site speed fast, you should choose the best file format and optimise image sizes and quality.

Image ALT tags - All images should use appropriate alt tags. it is not only good for search engines, but they are also good for accessibility.

3. Meta descriptions

It is important in getting users to click through from the search engine result page to website. You should write compelling meta descriptions and the recommended length is "150 to 160 characters".

4. Content with targeted keyword phrases

Create unique content on each page and it must be extremely reagent for that keyword phrase. When you choose keyword and phrases, you should find one which have a high search volume with little competition.

They are going to be picked from search engine and your site will come up the top of the searching result.

5. Make URLs SEO-Friendly

A good URL structure provides both users and search engines an idea of what the destination page is all about. Good URL is composed of short and keyword rich.

6. 404 error page

A 404 page, or error page, is the content a user sees when they try to reach a non-existent page on our website. It’s the page your server displays when it can’t find the URL requested by the user.
